# EthicApp Schemas

This directory currently contains JSON Schemas for portable EthicApp objects.

- `ethicapp-v1.schema.json`: legacy activity design schema.
- `case-document-representation-v1.schema.json`: canonical representation of a PDF-backed ethical case document as metadata, extracted raw text, and an ordered page-image sequence.

The directory name is historical. As schemas grow beyond activity designs, this should be renamed and organized as a broader canonical schema package.
